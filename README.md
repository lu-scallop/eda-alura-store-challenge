# Análise Exploratória (EDA) de vendas e desempenho das lojas do Sr. João

![Status do Projeto](https://img.shields.io/badge/status-desenvolvimento...-orange)

Projeto Challenge da trilha Data Science - Turma 9 do Programa ONE (AluraLatam + Oracle). Nesse desafio, o objetivo é analisar o desempenho de quatro lojas do Sr. João e determinar através das análises qual loja apresentou menor desempenho e portanto pode ser vendida. Para isso, estou utilizando quatro datasets individuais de cada uma das lojas. Desse modo o Sr. João poderá realizar um outro investimento com o capital obtido. O projeto ainda está em desenvolvimento. Essas são as métricas que estão sendo analisadas: 

* Faturamento
* Categorias mais vendidas
* Avaliações dos clientes
* Produtos mais vendidos e frete médio.
